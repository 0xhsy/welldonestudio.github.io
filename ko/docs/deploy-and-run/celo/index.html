Celo

Choose a wallet​

WELLDONE Code를 사용하기 위해서는 WELLDONE Wallet을 설치해야 합니다.

➡️ WELLDONE Wallet 설치하기

Connect​

WELLDONE wallet을 연결하기 이전에 WELLDONE wallet에 Celo account를 생성합니다.

Create Project​

Solidity로 컨트랙트를 작성합니다. 그리고 solidity로 작성된 컨트랙트 파일을 선택합니다.

Compile​

  • Solidity 컨트랙트 코드를 선택하면 Compile 버튼이 활성화됩니다.
  • Complie 버튼을 클릭합니다.

Deploy​

팁

WELLDONE Wallet에서는 해당 지갑 주소에 연결된 네트워크를 자동으로 감지하여 가져옵니다. 따라서 Deploy를 하기 이전에 메인넷에 트랜잭션을 보낼 것인지, 테스트넷에 트랜잭션을 보낼 것인지 미리 고려해두어야 합니다.

  • 컴파일된 컨트랙트 코드가 있는 경우 디플로이 버튼을 클릭합니다.
  • 지갑을 사용하여 거래에 서명합니다.

  • 컨트랙트가 배포되면, 컨트랙트 메소드를 실행할 수 있습니다.

View and Call function​

  • 위의 과정을 통해 컨트랙트를 배포하면 자동으로 배포된 컨트랙트가 호출됩니다.

  • 혹은 At Address 내부에 컨트랙트 아이디를 입력하고 At Address버튼을 눌러 기존에 배포된 컨트랙트를 호출할 수 있습니다.

  • 호출할 메소드를 선택합니다.

  • 필요에 따라 매개변수를 추가합니다.

  • call이나 transact를 통해 값을 읽거나 트랜잭션을 전송합니다.
